Create a virtual envicronment
Activate the environment

**Install the python packages**
```bash 
$ pip install -r requirements.txt
```

**To run the app**
```bash 
$ flask run
```

**To run the tests**
```bash 
$ pytest
```

The app should be running on port: 5000 by default.

**Features**
* API endpoint with validation
* Request caching
* Data Filtering
* Data Sorting
